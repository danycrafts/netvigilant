# Roadmap

This document outlines the future direction of the NetVigilant project.

## Phase 1: Stabilization & Refactor (Current)

- [x] Code Splitting, DRY Refactor, Tech Debt Fixes, and Performance/Caching Improvements.

## Phase 2: Core Feature Expansion

- [ ] **Authentication Implementation:** Integrate with a backend service for authentication.
- [ ] **Data Persistence:** Implement a local database to store historical network data.
- [ ] **UI/UX Enhancements:** Improve the user interface and user experience.

## Phase 3: Scaling & Optimization

- [ ] **Desktop/Web Support:** Add support for desktop and web platforms.
- [ ] **Advanced Analytics:** Provide more detailed analytics and insights into network usage.
- [ ] **Performance Optimization:** Further optimize the application for performance and battery life.
